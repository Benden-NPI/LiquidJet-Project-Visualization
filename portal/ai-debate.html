AI 辯論室

設定主題、角色與回合數，讓多個 AI 人格互相討論。

🔑 API Key 設定
Key 僅存於 localStorage，不會傳送到任何伺服器以外的地方。
支援 xAI API Key（以 xai- 開頭，可在 console.x.ai 取得；適用所有 Grok 系列，包含 grok-4）
或 GitHub PAT（以 github_pat_ / ghp_ 開頭，使用 GitHub Models；建立 PAT 時需勾選 Models → Read 權限）。填好後請點模型旁的「🔄 載入」，下拉選單只會顯示此 PAT 實際可呼叫的模型。
注意：GitHub Models 不提供 Anthropic Claude（Claude 走另一套 Messages API），故清單中通常沒有 claude-*。
如果已在編輯器將 Key 設定為 Webhook API Key，可點「從編輯器讀取」。
💬

在左側設定主題與角色後，
點選「▶ 開始辯論」啟動 AI 對話。
